Introduction

Blombly is a programming language with a reduced instruction set and algorithm-only semantics. This means that its instruction set is limited to a few but powerful operations that cover a broad range of scenarios. At the same time, the coder is asked to only design algorithms, with the language automating tasks like memory management and parallelization.
