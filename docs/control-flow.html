Control Flow

Control flow expressions alter which code segments are executed next. Blombly offers similar options to most programming languages in terms of condititional branching, loops, and method calling. The first two are descibed in this section, whereas method calling in section 2.4. However, the language also provides more dynamic options, namely the ability to "paste" code by diretly inlining it, which we introduce in next section, and success-error handling that intercepts return values, which is detailed in section 2.6. Intercepting success values can also create advanced versions of conditional branching and loops, and some patterns are described in this section for completeness.

Conditions

The if and while statements implement conditional branching and loops respectively. Of these, conditional statements evaluate expressions within the current context and execute the associated code block if the outcome is true. There may also be an optional else branch that is executed if the expression evaluates to false.

if (expression) {
    // code to be executed if expression is true
}
else {
    // code to be executed if expression is false
}

If code blocks comprise only one statement, you may replace brackets with that statement. In this case, a semicolon must be placed only at the end of the condition to separate it from the rest of code. If you accidentally pout a semicolon before an else statement, the compiler will create an error message for you. For example, the following code has valid syntax for both the first branch to print "positive" and the the nested condition that creates an else if clause. Given the option, avoid this type of nesting conditions, as they may create memory bloat. Instead, rely on try statements described below.

main.bb 
x = 1;
if (x>0) 
    print("positive")
else if(x<0)
    print("negative");

Loops

The while statement continuously executes a code block as long as the expression evaluates to true, where the expression is re-evaluated at each repetition. This creates a loop that can run indefinitely if the condition never becomes false. The syntax is as follows:

while (expression) {
    // code to be executed as long as expression is true
}

Here is an example where loop runs as long as the condition counter < 10 holds true. During each iteration, the loop prints the current value of the counter and increments it by 1. Once that value becomes 10, the condition evaluates to false, and the loop terminates.

main.bb 
counter = 0;
while (counter<10) {
    print("Counter is: " + str(counter));
    counter = counter + 1;
}

Introduction to signal interception

Blombly does not implement break and continue keywords, because these can be substituted by a mixture of the try and return statements. The latter create a type of signal interception pattern of those detailed in section 2.6; here, we get a first taste. The first is broadly responsible for intercepting any returning (or error) values so that they can be assigned to variables, whereas the second for stopping code execution with a returning value. The usage associated with code execution is to use try before control flow statements so as to assign their returning outcome. As a simple example, here is an one-liner that computes the sign of a variable x.

sgn = try if(x>=0) return 1 else return -1;

Use a similar syntax to break from loops, though handling missing return values will not be described for now. What should be mentioned is that, constrary to errors, intercepting returning values is lightweight. 

counter = 0;
try while (true) {
    print("Counter is: " + str(counter));
    counter = counter + 1;
    if(counter==10)
        return;
}

Finally, prepend try to the loop body's opening bracket to intercept return values there. Notice that this syntax only allows one among breaking, continuing, or triggering an unconstrained return (e.g., that would return from a method) in the same loop. This helps write simple code that is easy to understand by bringing only one modification to each loop. 

counter = 0;
while (counter<10) try {
    if (counter % 2==0)
        return;
    print("Odd value: " + str(counter));
    counter = counter + 1;
}
